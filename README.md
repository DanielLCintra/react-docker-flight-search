# MaxMilhas Frontend test - by Daniel Cintra

## To run project in development mode: 

```sh
yarn run docker-dev
```

## To build project 

```sh
yarn run docker-build
```

## To run tests 

```sh
yarn run docker-tests
```

## Utilized Libraries

To develop the test were used:

* React
* Redux
* React Router
* Lodash
* Axios
* Moment 
* React Flexbox Grid
* React Feather
* React Datetime
* React Click Outside
* ValidatorJS
* Debounce
* Docker
