# MaxMilhas Frontend test - by Daniel Cintra

##To run project in development mode: 

yarn run docker-dev

##To build project 

yarn run docker-build

##To run tests 

yarn run docker-tests

To develop the test were used:

* React
* Redux
* React Router
* Lodash
* Axios
* Moment 
* React Flexbox Grid
* React Feather
* React Datetime
* React Click Outside
* ValidatorJS
* Debounce
* Docker
